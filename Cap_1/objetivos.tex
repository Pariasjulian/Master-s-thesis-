\section{Objectives}

\subsection{General Objective}
To develop an EEG-based data acquisition framework to mitigate mixed-signal interference and bound communication latency, ensuring support for neurocognitive assessments in ADHD monitoring.

\subsection{Specific Objectives}
\begin{enumerate}
    \item To design and construct an EEG cap that overcomes physical deployment barriers and maximizes signal efficiency through optimized electrode-scalp coupling and systematic impedance measurement.
    \item To develop an embedded system-based EEG acquisition portable device that guarantees high-fidelity, low-noise signal processing for continuous, high-frequency data logging.
    \item To implement a concurrent processing architecture that ensures precise temporal synchronization for ADHD monitoring by leveraging direct hardware-level event alignment, validated against end-to-end latency and jitter thresholds.
\end{enumerate}
